# ServiNow – Local Services Marketplace (MERN)

ServiNow is a full-stack MERN application that connects users with
trusted local service professionals such as plumbers, electricians,
cleaners, and salon experts.

## Features
- Browse services and professionals
- Live availability of workers
- Transparent pricing & ratings
- Service-wise professional listing
- Clean, scalable MERN architecture

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Architecture: REST API

## Project Structure
servinow/
├── client/   # React frontend
└── server/   # Node/Express backend

## How to Run
npm run dev
